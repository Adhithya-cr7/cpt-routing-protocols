This repository contains the Cisco Packet Tracer (`.pkt`) files and documentation for configuring dynamic routing protocols across a multi-router network topology. 

## Objective
To configure, verify, and test the following routing protocols between multiple routers to ensure end-to-end communication between different networks:
* **OSPF** (Open Shortest Path First)
* **RIPv2** (Routing Information Protocol Version 2)
* **EIGRP** (Enhanced Interior Gateway Routing Protocol)

## Network Topology
The network consists of:
* 3 Routers 
* 2 Switches
* 4 End Devices (PCs)

## Files Included
* network-routing.pkt 
* `Routing_Lab_Report.pdf` - Detailed procedure, IP configurations, and CLI verification screenshots.

## Verification Commands Used
* `show ip interface brief`
* `show ip route`
* `show ip ospf neighbor`
* `show ip protocols`
* `show ip eigrp neighbors`
